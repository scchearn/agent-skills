# close-off Skill

## Purpose

Graceful session wrap-up with status capture and time tracking.

## Trigger

When the user says something like:
- "I'm done for now"
- "Let's wrap up"
- "Close off"
- "Finished"

## Workflow

1. **Summarize** — what was accomplished
2. **Capture** — any pending decisions or blockers
3. **Update** — TaskWarrior (completions, new tasks)
4. **Time** — stop Clockify timer (see [amendment](./close-off-amendment.md))
5. **Confirm** — ready to close
